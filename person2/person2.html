Project 2 : parallax

เว็บไซต์ที่ใช้เทคนิค Parallax Scrolling เพื่อสร้างเอฟเฟกต์การเลื่อนภาพแบบมีมิติ เพิ่มความน่าสนใจให้กับหน้าเว็บ

Description

โปรเจกต์นี้เป็นเว็บไซต์ที่พัฒนาโดยใช้เทคนิค Parallax Scrolling ซึ่งทำให้พื้นหลังและองค์ประกอบต่าง ๆ เคลื่อนไหวด้วยความเร็วที่แตกต่างกันขณะเลื่อนหน้าเว็บ ช่วยสร้างความรู้สึกมีมิติและเพิ่มประสบการณ์การใช้งานให้ดูน่าสนใจมากยิ่งขึ้น โครงสร้างเว็บไซต์ถูกออกแบบให้เรียบง่าย เน้นการจัดวางภาพและข้อความให้สอดคล้องกับการเลื่อนหน้าเว็บ ใช้ HTML และ CSS ในการจัดโครงสร้างและออกแบบสไตล์ พร้อมเสริมด้วย JavaScript เพื่อควบคุมเอฟเฟกต์การเลื่อน เหมาะสำหรับการเรียนรู้พื้นฐานของ Parallax Effect และการพัฒนาเว็บไซต์เชิง Interactive
